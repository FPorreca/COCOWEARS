# SSVEP-Based Brain-Computer Interface Framework

This repository presents a complete framework for the acquisition, analysis, training, and real-time execution of EEG-based Brain-Computer Interface (BCI) systems using Steady-State Visual Evoked Potentials (SSVEPs). The project is based on experiments performed with the BitBrain DIADEM EEG headset and includes both offline and real-time processing pipelines for Machine Learning (ML) and Deep Learning (DL) applications.

The repository is organized into four main modules covering:

- EEG signal acquisition and preprocessing
- Frequency-domain and PSD analysis
- ML/DL model training and validation
- Real-time EEG classification on desktop and embedded devices

# Repository Structure

## 1.SSVEP_PSD_Analysis

This folder contains the frequency-domain analysis of EEG signals acquired during SSVEP experiments using the BitBrain DIADEM headset.

The EEG data were collected through:

- BitBrain Viewer
- OpenViBE

as described in the associated research papers.

The Python scripts included in this module perform:

- EEG signal preprocessing
- Band-pass and notch filtering
- Power Spectral Density (PSD) computation
- Bandpower analysis for each EEG channel
- Frequency-domain visualization of SSVEP responses

### output_examples

This subfolder contains example outputs generated by the analysis pipeline, including:

- Bandpower plots for different EEG channels
- PSD visualizations for the O2 channel
- Comparisons between different SSVEP target frequencies
- CSV files containing PSD values for each EEG channel and experiment step

---

## 2.SSVEP_Model_Training

This folder contains Python scripts for training and validating Machine Learning and Deep Learning models for SSVEP classification.

The implemented pipeline includes:

- EEG preprocessing
- Sliding-window extraction
- PSD feature extraction using Welch’s method
- Dataset generation
- Model training and evaluation

The implemented classifiers include:

- Linear Discriminant Analysis (LDA)
- Support Vector Machine (SVM)
- Decision Tree (DT)
- Random Forest (RF)
- K-Nearest Neighbors (KNN)
- Convolutional Neural Network (CNN)

The training and validation procedures follow the methodology described in the publications above and are designed for both subject-specific and aggregated EEG datasets.

---

## 3.SSVEP_Acquisition_GUI_LSL

This folder contains the Python application developed for real-time SSVEP experiments using the Lab Streaming Layer (LSL) protocol.

The application:

- Generates the SSVEP stimulation GUI and displays flickering visual targets at different frequencies
- Acquires EEG signals in real time from the DIADEM headset through LSL
- Applies previously trained ML/DL models to incoming EEG data
- Performs real-time SSVEP classification

---

## 4.SSVEP_Realtime_MQTT

This folder contains the embedded real-time implementation developed for Raspberry Pi devices.

The application:

- Acquires EEG signals directly from the BitBrain DIADEM headset via Bluetooth
- Executes trained ML models in real time
- Performs online SSVEP recognition
- Publishes classification results through the MQTT protocol

This module represents the edge-computing component of the proposed Cloud-Edge BCI architecture and is intended for portable real-time cognitive environment applications.
